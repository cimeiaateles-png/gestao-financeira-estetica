# Sistema de Gestão Financeira para Profissional Autônoma da Área da Estética

Aplicação web desenvolvida para auxiliar no controle financeiro de um pequeno negócio do setor de estética, automatizando cálculos de faturamento, comissões e organização de dados.

---

# Sobre o projeto

Este projeto foi desenvolvido para atender às necessidades reais de uma profissional autônoma da área da estética.

A cliente realiza procedimentos estéticos em uma clínica parceira, onde utiliza salas alugadas por período para atender seus pacientes. Como não possuía uma ferramenta para organizar suas informações financeiras, surgiu a necessidade de desenvolver um sistema que centralizasse o controle do negócio.

O objetivo da aplicação é facilitar o acompanhamento financeiro, permitindo registrar procedimentos, organizar receitas e oferecer uma visão mais clara dos resultados da atividade profissional.

---

# Problema

Profissionais autônomos da área da estética muitas vezes realizam o controle financeiro de forma manual, utilizando planilhas ou anotações, o que aumenta o risco de erros, dificulta a organização das informações e torna a análise financeira mais trabalhosa.

---

# Solução

Desenvolvimento de uma aplicação web para centralizar o controle financeiro da profissional, automatizando cálculos e organizando as principais informações do negócio de forma simples e intuitiva.
